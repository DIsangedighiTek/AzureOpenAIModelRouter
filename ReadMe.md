# Model Router Prompt Evaluation

This repository contains a small test client for evaluating prompt behavior against the Azure OpenAI model router.

## Files

- [testClient.py](testClient.py): Python script that sends a system/user prompt to the Azure OpenAI endpoint and prints the result.
- [ReadMe.md](readMe.md): User Guide
